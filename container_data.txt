# Real-Time Container Fill Tracker - Data Log and Methodology  

## 1. SENSOR SETUP AND MEASUREMENT METHOD  

- **Sensor Used:** HC-SR04 Ultrasonic Sensor  
- **Controller:** NodeMCU (ESP8266)  
- **Measurement Principle:**  
  `Distance = (Time × Speed of Sound) / 2`  

- **Container Height:** 40 cm (empty)  
- **Fill Threshold:** 75% (Alert triggered at ~30 cm filled)  

Sensor is mounted on the container lid; it measures distance from the sensor to the material/waste surface.  

---

## 2. HOW FILL LEVEL IS CALCULATED  

**Formula:**  
Distance_to_material = Sensor Reading (cm)
Fill Level (%) = ((Container_Height - Distance_to_material) / Container_Height) × 100

**Example Calculation:**  
- Sensor Reading = 5.2 cm  
- Fill Level = `((40 - 5.2) / 40) × 100 = 87.0%`
